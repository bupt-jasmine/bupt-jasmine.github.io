# 如何添加笔记

## C++ 学习笔记

在 `_cpp/` 目录下创建新的 Markdown 文件，格式如下：

```markdown
---
title: "笔记标题"
collection: cpp
permalink: /cpp/your-note-name
excerpt: '简短描述'
date: 2025-12-02
---

## 你的内容

在这里写你的学习笔记...

```cpp
// 代码示例
```

---

**笔记日期**: 2025-12-02
```

**文件命名规则**: `YYYY-MM-DD-文件名.md`，例如 `2025-12-02-smart-pointers.md`

## PyTorch 学习笔记

在 `_pytorch/` 目录下创建新的 Markdown 文件，格式如下：

```markdown
---
title: "笔记标题"
collection: pytorch
permalink: /pytorch/your-note-name
excerpt: '简短描述'
date: 2025-12-02
---

## 你的内容

在这里写你的学习笔记...

```python
# 代码示例
import torch
```

---

**笔记日期**: 2025-12-02
```

**文件命名规则**: `YYYY-MM-DD-文件名.md`，例如 `2025-12-02-neural-networks.md`
