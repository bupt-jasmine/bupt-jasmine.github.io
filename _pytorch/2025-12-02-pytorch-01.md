这里介绍如何在 macOS 上从源码编译和安装 PyTorch。

Windows 和 Linux 系统的安装请参考官方文档。

## 环境准备

### Miniconda 安装

```bash
mkdir -p ~/miniconda3
curl https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh -o ~/miniconda3/miniconda.sh
bash ~/miniconda3/miniconda.sh -b -u -p ~/miniconda3
rm ~/miniconda3/miniconda.sh
```

创建一个新的 conda 环境用于 PyTorch 编译和测试：

```bash
conda create -n torch python=3.11
```

激活环境：

```bash
conda activate torch
```

## PyTorch 源码编译安装

这里使用的是 PyTorch 2.8.0 版本作为示例（以后的源码解读也是以这个版本的源码为例）。

克隆 PyTorch 源码：

```bash
git clone -b v2.8.0 --recursive https://github.com/pytorch/pytorch.git

cd pytorch
```

安装编译依赖：

```bash
conda install cmake ninja

# 支持分布式需要安装 libuv
conda install pkg-config libuv

pip install -r requirements.txt
``` 

设置编译选项并编译安装：

```bash
DEBUG=1 USE_DISTRIBUTED=1  USE_MKLDNN=0 USE_CUDA=0 BUILD_TEST=0 USE_FBGEMM=0 USE_NNPACK=0 USE_QNNPACK=0 USE_XNNPACK=0 python setup.py develop
```

这里开启 DEBUG 模式，方便后续调试源码。

编译完成后，打印版本信息查看是否安装成功：
```python
>>> import torch
>>> print(torch.__version__)
2.8.0a0+gitba56102
>>>
```
