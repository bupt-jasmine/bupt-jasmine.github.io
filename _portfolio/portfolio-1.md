## 项目简介

设计并实现了一个高效的数据处理框架，采用 C++/Python 混合开发，基于 Bazel 构建系统。

## 核心功能

- 高效的数据流水线处理
- 批处理和流式处理能力
- 数据读取、解压缩、文本解析、数据过滤等可复用组件
- 通过 pybind11 实现 Python 接口封装

## 技术栈

- C++
- Python
- Bazel
- pybind11

## 应用场景

应用于机器学习训练数据预处理场景，提升了数据处理效率。
